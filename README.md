# ROBOHUB 机研社

面向机器人开发者的精选资讯、工程实践、开源项目与技术社区。网站参考信息流产品的高密度阅读方式，但品牌、内容与交互均围绕机器人开发重新设计。

## 功能

- 精选、全部动态、机器人日报、技术主题和开源项目导航
- 具身智能、仿真、控制、硬件、开源分类筛选
- 标题、摘要、标签与来源全文搜索
- 设备本地收藏与收藏视图
- 桌面、平板和手机响应式布局
- Agent 接入、关于、更新日志与反馈页面状态

## 本地开发

```bash
npm ci
npm run dev
```

## 验证

```bash
npm run lint
npm test
```

## 内容原则

社区优先收录可复现、可测量、有明确证据边界的机器人开发内容。研究结论、工程经验和产品判断应清楚区分，硬件参数必须保留测试条件。
